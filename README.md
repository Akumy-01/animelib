# animelib

帮你记录管理动漫图书馆。

## 功能

- 支持动画条目的列表、网格和画廊视图。
- 支持 TMDb 搜索、详情同步、海报更换、角色、制作人员和集数信息。
- 支持观看状态、评分、收藏、备注和 JSON 数据导入导出。
- Windows 桌面应用，基于 Tauri + React。

## 安装包

最新 Windows 安装包在 `release/` 目录：

- `animelib_0.1.3_x64-setup.exe`
- `animelib_0.1.3_x64_en-US.msi`

## 开发

```powershell
npm install
npm run tauri -- dev
```

## 打包

```powershell
npm run tauri -- build
```
